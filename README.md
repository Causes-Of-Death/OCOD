# OCOD
OCOD: Ontology of Causes of Death

The OCOD ontology is developed to standardize and formalize the language used in medicolegal death certification. Cause-of-death statements as recorded by medical examiners and coroners vary substantially in terminology and structure, creating barriers to data interoperability, public health surveillance, and epidemiologic research. OCOD provides a formal OWL-based knowledge representation framework that classifies death certification terminology into structured, queryable entities using description logic reasoning.

OCOD is intended to support automated classification of free-text cause-of-death language, improve consistency across jurisdictions, and enable downstream public health applications. The ontology aims to support property-based classification to capture the multifactorial nature of forensic death certification.

All ontology content is freely available and open access. OCOD is intended as a community resource for forensic pathologists, toxicologists, epidemiologists, vital records personnel, and informatics researchers working with death certification data.
